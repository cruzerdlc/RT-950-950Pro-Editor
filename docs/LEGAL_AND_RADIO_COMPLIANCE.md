# Legal and Radio Compliance Notes

This project is a programming tool. It does not grant permission to transmit.

Users are responsible for:

- using only frequencies they are licensed or otherwise authorized to use,
- following local, state, federal, and country-specific radio regulations,
- using legal power levels and bandwidths,
- not transmitting on restricted, emergency, aviation, marine, public safety, or commercial frequencies without authorization,
- keeping backups before writing a radio.

Some stock presets may be receive-only or require specific licenses. Review each channel before transmitting.

This project is not affiliated with Radtel.
