# Antivirus false-positive notes

This is a free community radio editor. The Windows executable may be flagged by Microsoft Defender or another antivirus product because it is currently unsigned and has limited reputation.

A detection does not automatically prove the app is malicious, but users should still be cautious:

- Download only from the official project release page.
- Do not run copies from unknown sources.
- Keep backups of radio data.
- Do not permanently disable antivirus protection.
- Submit the exact detected file to the antivirus vendor as a false positive if you believe the detection is incorrect.

The public app no longer includes hidden developer packet-capture tools or Virtual COM lab tools.
