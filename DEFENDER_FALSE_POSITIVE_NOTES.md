# Microsoft Defender false-positive notes

This is a free community project. The released Windows executable may still be flagged by Microsoft Defender because it is unsigned and has limited reputation.

The public app does not include the hidden developer packet-capture tools, Virtual COM lab tools, or boot-image capture lab tools that were previously used during development.

Users should download only from the official release page. If Microsoft Defender incorrectly flags the app, submit the exact detected file to Microsoft Security Intelligence as a false positive.

Code signing would improve reputation, but paid signing is not currently practical for this free project.
